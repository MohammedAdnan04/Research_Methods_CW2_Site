# NHS COVID-19 Emergency Admissions Dashboard

This repository contains the interactive data visualization website for **COMP4037 Research Methods Coursework 2**, created by **Mohammed Adnan**.

## 📊 About the Project
The project investigates the following research question:
> *Which NHS emergency hospital admission categories were most suppressed by the COVID-19 lockdown (2020/21), and did they recover to pre-pandemic levels by 2022/23?*

The interactive dashboard features an **Annotated Diverging Heatmap** that explores the year-on-year percentage change in NHS Emergency Admissions across 18 ICD-10 diagnostic chapters compared to the 2018/19 pre-pandemic baseline.

## ✨ Features
- **Interactive Heatmap:** Built with Python (Plotly) and embedded into a modern, responsive website.
- **Dynamic Filtering:** Includes dropdowns to sort by lockdown impact, filter by drop thresholds, and focus on specific financial years.
- **Detailed Tooltips:** Hover over any data point to reveal specific admission counts, percentage changes, baseline figures, and top affected age groups.
- **Full Research Report:** Detailed breakdown of visual mappings, variables, and data preparation techniques.

## 🚀 Technologies Used
- **Data Processing & Visualization:** Python, Pandas, Plotly
- **Web Frontend:** HTML5, Vanilla CSS (Dark Mode/Glassmorphism), Vanilla JavaScript
